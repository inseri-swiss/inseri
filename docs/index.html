inseri vision

inseri connects data providers, web components, local user data and microservices,
so that a user can combine them in any possible way


⇩

web components

inseri can integrate any own or third party web component as an inseri app.
Any inseri app can be used with own user data, existing data providers and microservices


Real world example: """ During my phd I created a web component that I would like to publish in a way that users without programming knowledge can use it """
