# Radial Bar Chart
The Radial Bar Chart app renders a basic radial bar chart based on JSON data input.

## Features
- Tooltip indicating each bar's value

## JSON Data Input

The Radial Bar Chart app works with a JSON data input as follows: 

```js
"data": {
  "data": [
    {
      "label": "1",
      "value": 179
    },
    {
      "label": "2",
      "value": 38
    },
    {
      "label": "3",
      "value": 77
    },
    {
      "label": "4",
      "value": 70
    },
    {
      "label": "5",
      "value": 78
    },
    {
    .
    .
    .
    .
  ]
}
```
